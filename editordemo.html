Stylotron Editors Demo

Editor #1 These editors dynamically mark up content as its edited using lib/stylotron.js. The main demo (stylotron-demo.html) explains Stylotron's markup generation process. The patterns (stylotron-samplepatterns.js) and styles (stylotron-samplestyles.css) used here are overviewed in the IDE's (stylotronIDE.html) introductory document. Texteditor.js (lib/texteditor.js) provides the basic editor and stylotroneditor.js (lib/stylotroneditor.js) provides an attachment that adds dynamic markup capability to it. These libraries also provide undo/redo, search, and souped-up selections support. This editor makes use of the default selections attachment, which keeps a highlight on the current selection when the element loses focus. The call that creates and initializes this editor is: UserDocument.editor1 = UI.stylotroneditor.createinit( 'editor1', {selections: {}}, UserDocument.patternseries ); Editors operate autonomously once they are initialized. Editor #2 In addition to a selections attachment, this editor makes use of undoredo and search attachments, which manage the controls below and their events. The call that creates and initializes this editor is: UserDocument.editor2 = UI.stylotroneditor.createinit( 'editor2', {selections: {}, undoredo: {}, search: {}}, UserDocument.patternseries );
